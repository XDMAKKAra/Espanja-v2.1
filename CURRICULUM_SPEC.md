# Puheo — Pedagoginen Curriculum Spec

> Kirjoitettu 2026-04-28. Tämä on source-of-truth kaikelle oppimispolku-kehitykselle.
> Agentti: lue tämä ENNEN kuin kirjoitat yhtään riviä koodia L-PLAN-looppeihin.

---

## 1. Filosofia

Puheo on **yksityinen tutori**, ei oppikirja. Se tietää missä oppilas on, mihin hän tähtää,
ja puhuu hänelle suoraan suomeksi. Kaikki sisältö on ankkuroitu yhteen päivämäärään: **28.9.2026**.

Kolme periaatetta:
1. **Polku ensin, vapaus aina** — oppilas saa selvän suunnitelman oletuksena, mutta voi tehdä
   mitä tahansa vasemman navin kautta milloin tahansa.
2. **Taso nousee suorituksesta, ei ajasta** — nopea oppija pääsee eteenpäin nopeasti.
   Hidas oppija ei putoa pois — hän harjoittelee enemmän samalla tasolla.
3. **Jokainen oppilas oppii KAIKEN** — ei ole "E-tason oppilas joka ei tarvitse subjunktiivia".
   Arvosanatavoite määrää VAUHDIN ja PAINOTUKSEN, ei sisällön kattavuutta.

## 2. YO-arvosanatavoite → oppimisjärjestys

### Mitä YO-koe mittaa (lyhyt espanja, EAB2)
- Kuullun ymmärtäminen (ei Puheossa — ei äänitehtäviä)
- Luetun ymmärtäminen (~25% pisteistä)
- Rakenteiden hallinta / aukkotehtävät (~25%)
- Kirjoittaminen — lyhyt + pitkä teksti (~50%)

Kirjoittaminen on AINA isoin osa. Kaikki oppilaat tarvitsevat kirjoittamista riippumatta tavoitteesta.

### Arvosanatavoite → priorisointi (EI sisältörajoitus)

| Tavoite | Painotus | Nopeus | Vaikein asia jonka on hallittava |
|---------|----------|--------|----------------------------------|
| **I/A** | Sanasto + preesens + preteriti. Kirjoittaminen: perusvastaus. | Hidas, paljon toistoa | Preteriti vs imperfekti |
| **B** | Yllä + imperfekti + konditionaali. Kirjoittaminen: rakenne kunnossa. | Normaali | Aikamuotojen sekoittaminen luontevasti |
| **C** | Yllä + futuuri + konditionaali + perus-subjunktiivi. Kirjoittaminen: sidosteisuus. | Normaali | Ojalá / es importante que |
| **M** | Yllä + subjunktiivi syvemmin + pluskvamperfekti. Kirjoittaminen: argumentointi. | Nopea | Subjunktiivi imperfekti (si tuviera) |
| **E** | Kaikki rakenteet. Kirjoittaminen: rekisteri + sidosteisuus + tyyli. | Nopea | Kaikki subjunktiivimuodot + pluskvamp. |
| **L** | Kuten E, mutta täydellinen kontrolli. Harvinainen rakenteet hallinnassa. | Erittäin nopea | Ei lisäsisältöä — syvempi automatisointi |

**Tärkeää:** E-tavoitteinen oppilas oppii subjunktiivin. M-tavoitteinen oppii pluskvamperfektin.
Ero on siinä, kuinka paljon harjoittelua kuhunkin käytetään ja milloin se tulee polulla.

## 3. Kurssirakenne — 8 kurssia

Rakenne noudattaa lukion EAB2-kurssien henkeä mutta ei kopioi sisältöä.
Jokaisessa kurssissa on **sanasto + kielioppi + kirjoittaminen yhdistettynä** —
ei "tämä on sanastokurssi" tai "tämä on kielioppikurssi".

### Kurssi 1 — "Kuka olen" (A-taso, avoin kaikille)
**Kielioppi:** Preesens säännölliset (-ar/-er/-ir), ser vs estar perusteet, numerot
**Sanasto:** Persoona, perhe, koulu, värit, kansallisuudet
**Kirjoittaminen:** Lyhyt esittelyteksti (50–80 sanaa)
**Oppitunteja:** 10
**Avautuu:** Aina — tämä on lähtöpiste kaikille

Oppituntirakenne:
1. Sanasto: perhe + kansallisuudet (MCQ)
2. Kielioppi: -ar verbit preesensissä (aukkotehtävä)
3. Kielioppi: -er/-ir verbit (aukkotehtävä)
4. Sanasto: koulu + värit (MCQ)
5. Yhdistelmä: ser vs estar intro (aukkotehtävä)
6. Sanasto: numerot + ikä (MCQ)
7. Kielioppi: ser/estar syvemmin (aukkotehtävä)
8. Luetun ymmärtäminen: lyhyt teksti perheestä (A-taso)
9. Kirjoittaminen: kirjoita itsestäsi (50–80 sanaa, ei YTL-rubriikki vielä)
10. Kertaustesti: 15 kysymystä kurssista → ≥80% = kurssi suoritettu

### Kurssi 2 — "Arki ja elämä" (A-taso)
**Kielioppi:** Preesens epäsäännölliset (ser/estar/tener/ir/hacer), gustar-rakenne
**Sanasto:** Koti, ruoka, vapaa-aika, harrastukset, viikonpäivät/kuukaudet
**Kirjoittaminen:** Sähköposti ystävälle arkipäivästä (80–120 sanaa)
**Oppitunteja:** 10

Oppituntirakenne:
1. Sanasto: koti + huonekalut (MCQ)
2. Kielioppi: ser/estar/tener/ir epäsäännölliset (aukkotehtävä)
3. Sanasto: ruoka + ateriat (MCQ)
4. Kielioppi: gustar + indirekti pronominit (aukkotehtävä)
5. Sanasto: vapaa-aika + harrastukset (MCQ)
6. Kielioppi: hacer/poder/querer + stem-changers (aukkotehtävä)
7. Yhdistelmä: arjen kuvaus (sekamuoto)
8. Luetun ymmärtäminen: arki-aiheinen teksti (A-taso)
9. Kirjoittaminen: sähköposti + gustar-rakenne (80–120 sanaa)
10. Kertaustesti → ≥80% = suoritettu

### Kurssi 3 — "Mitä tein" (B-taso)
**Kielioppi:** Preteriti (säännölliset + yleisimmät epäsäännölliset: ser/ir/tener/estar/hacer)
**Sanasto:** Matkustaminen, liikenne, hotelli, matkailu
**Kirjoittaminen:** Kertomus lomamatkasta (100–150 sanaa)
**Oppitunteja:** 11

Oppituntirakenne:
1. Sanasto: matkustaminen + kulkuvälineet (MCQ)
2. Kielioppi: preteriti -ar verbit (aukkotehtävä)
3. Kielioppi: preteriti -er/-ir verbit (aukkotehtävä)
4. Sanasto: hotelli + majoitus (MCQ)
5. Kielioppi: preteriti epäsäännölliset ser/ir (aukkotehtävä)
6. Kielioppi: preteriti epäsäännölliset tener/estar/hacer (aukkotehtävä)
7. Sanasto: matkailu + nähtävyydet (MCQ)
8. Yhdistelmä: preteriti + matkailusanasto (sekamuoto)
9. Luetun ymmärtäminen: matkablogi (B-taso)
10. Kirjoittaminen: kertomus lomasta — preteriti pakollinen (100–150 sanaa)
11. Kertaustesti → ≥80% = suoritettu

### Kurssi 4 — "Ennen ja nyt" (B-taso)
**Kielioppi:** Imperfekti (säännölliset + ser/ir/ver), preteriti VS imperfekti (YO-klassikko)
**Sanasto:** Lapsuus, muistot, eläimet ja luonto, adjektiivit
**Kirjoittaminen:** Lapsuuden kuvaus TAI vertailuteksti (120–160 sanaa)
**Oppitunteja:** 12

Oppituntirakenne:
1. Sanasto: lapsuus + muistot (MCQ)
2. Kielioppi: imperfekti -ar verbit (aukkotehtävä)
3. Kielioppi: imperfekti -er/-ir + epäsäännölliset (aukkotehtävä)
4. Sanasto: eläimet + luonto (MCQ)
5. Kielioppi: preteriti vs imperfekti — perusero (aukkotehtävä)
6. Kielioppi: pret. vs imperf. — keskeytys (estaba + cuando) (aukkotehtävä)
7. Kielioppi: pret. vs imperf. — toistuvuus vs kertaluontoisuus (aukkotehtävä)
8. Sanasto: adjektiivit + kuvailut (MCQ)
9. Yhdistelmä: lapsuuden tarina (sekamuoto)
10. Luetun ymmärtäminen: nostalgia-aiheinen teksti (B-taso)
11. Kirjoittaminen: lapsuuden kuvaus — molemmat aikamuodot (120–160 sanaa)
12. Kertaustesti → ≥80% = suoritettu

### Kurssi 5 — "Tulevaisuus ja unelmat" (C-taso)
**Kielioppi:** Futuuri (säännölliset + epäsäännölliset), konditionaali, futuro próximo (ir a + inf)
**Sanasto:** Työ ja ura, unelmat, teknologia, yhteiskunta perusteet
**Kirjoittaminen:** Tulevaisuuden suunnitelmat TAI hypoteettinen tilanne (140–180 sanaa)
**Oppitunteja:** 11

Oppituntirakenne:
1. Sanasto: työ + ammatit (MCQ)
2. Kielioppi: futuuri säännölliset (aukkotehtävä)
3. Kielioppi: futuuri epäsäännölliset (tendré/pondré/haré/diré) (aukkotehtävä)
4. Sanasto: teknologia + internet (MCQ)
5. Kielioppi: futuro próximo (ir a + inf) (aukkotehtävä)
6. Kielioppi: konditionaali säännölliset (aukkotehtävä)
7. Kielioppi: konditionaali epäsäännölliset (aukkotehtävä)
8. Yhdistelmä: tulevaisuuden kuvaus (sekamuoto)
9. Luetun ymmärtäminen: artikkeli tulevaisuudesta (C-taso)
10. Kirjoittaminen: kirje tulevaisuudesta — futuuri + konditionaali (140–180 sanaa)
11. Kertaustesti → ≥80% = suoritettu

### Kurssi 6 — "Maailman ongelmat" (C-taso)
**Kielioppi:** Subjunktiivi preesens INTRO (ojalá, es importante/necesario/bueno que, quiero que)
**Sanasto:** Ympäristö, ilmastonmuutos, yhteiskunta, politiikan perussanasto
**Kirjoittaminen:** Mielipidekirjoitus yhteiskunnallisesta aiheesta (160–200 sanaa)
**Oppitunteja:** 12

Oppituntirakenne:
1. Sanasto: ympäristö + luonto (MCQ)
2. Kielioppi: subjunktiivin muodostus — säännölliset (aukkotehtävä)
3. Kielioppi: subjunktiivin muodostus — epäsäännölliset (aukkotehtävä)
4. Sanasto: yhteiskunta + politiikka (MCQ)
5. Kielioppi: ojalá + subjunktiivi (aukkotehtävä)
6. Kielioppi: es importante/necesario/bueno que (aukkotehtävä)
7. Kielioppi: quiero/espero/prefiero que (aukkotehtävä)
8. Sanasto: ilmastonmuutos + ympäristöongelmat (MCQ)
9. Yhdistelmä: mielipide subjunktiivin kanssa (sekamuoto)
10. Luetun ymmärtäminen: ympäristöaiheinen artikkeli (C-taso)
11. Kirjoittaminen: mielipidekirjoitus — subjunktiivi pakollinen (160–200 sanaa)
12. Kertaustesti → ≥80% = suoritettu

### Kurssi 7 — "Kulttuuri ja yhteiskunta" (M-taso)
**Kielioppi:** Subjunktiivi syvemmin (kun/antes de que/para que/no creer que/dudar que),
             pluskvamperfekti (había + part.), passiivin pintakosketus
**Sanasto:** Kulttuuri, taide, historia, media, monimutkaisempi yhteiskuntasanasto
**Kirjoittaminen:** Argumentoiva teksti monimutkaisemmasta aiheesta (180–220 sanaa)
**Oppitunteja:** 12

Oppituntirakenne:
1. Sanasto: kulttuuri + taide (MCQ)
2. Kielioppi: subjunktiivi aikaa ilmaisevien konjunktioiden kanssa (cuando/antes de que/después de que) (aukkotehtävä)
3. Kielioppi: subjunktiivi tarkoitusta ilmaisevien kanssa (para que/a fin de que) (aukkotehtävä)
4. Sanasto: historia + politiikka (MCQ)
5. Kielioppi: subjunktiivi kieltoa ilmaisevien kanssa (no creer/dudar/negar) (aukkotehtävä)
6. Kielioppi: pluskvamperfekti — muodostus (aukkotehtävä)
7. Kielioppi: pluskvamperfekti — käyttö (ennen toista menneisyyttä) (aukkotehtävä)
8. Sanasto: media + kommunikaatio (MCQ)
9. Yhdistelmä: kulttuuri + molemmat subjunktiivirakenteet (sekamuoto)
10. Luetun ymmärtäminen: kulttuuriaiheinen teksti (M-taso)
11. Kirjoittaminen: argumentoiva teksti — pluskvamp. + subjunktiivi (180–220 sanaa)
12. Kertaustesti → ≥80% = suoritettu

### Kurssi 8 — "YO-koevalmiiksi" (E/L-taso)
**Kielioppi:** Subjunktiivi imperfekti (si tuviera... / quisiera), kertaus kaikista rakenteista
**Sanasto:** Kaikki teemat yhdistettynä — YO-koetyyppinen sanasto
**Kirjoittaminen:** Täysmittainen YO-tyyppinen kirjoitustehtävä (200–240 sanaa)
**Oppitunteja:** 12

Oppituntirakenne:
1. Kielioppi: subjunktiivi imperfekti — muodostus (aukkotehtävä)
2. Kielioppi: subjunktiivi imperfekti — si-lauseet (si tuviera, haría) (aukkotehtävä)
3. Sanasto: YO-koetyyppinen sekatesti kaikista teemoista (MCQ)
4. Kielioppi: kertaus — kaikki aikamuodot sekalaisissa lauseissa (aukkotehtävä)
5. Yhdistelmä: pitkä aukkotehtävä (10+ aukkoa, kaikki rakenteet)
6. Luetun ymmärtäminen: YO-tasoinen teksti (E-taso)
7. Kirjoittaminen: lyhyt teksti YO-aiheella (160–200 sanaa)
8. Luetun ymmärtäminen: toinen YO-tasoinen teksti
9. Kielioppi: subjunktiivi imperfekti — harjoittelu sidosteisuussanoilla
10. Kirjoittaminen: pitkä YO-tyyppinen teksti (200–240 sanaa)
11. Täyskoe-simulaatio: luettu + rakenteet + kirjoittaminen (ajastettu)
12. Loppukertaus + henkilökohtainen palaute tutorilta

## 4. Etenemissäännöt

### Avautuminen
- Kurssi 1: aina avoin
- Kurssi N+1: avautuu kun Kurssi N:n kertaustesti ≥80%
- **Poikkeus — sijoitustesti**: Jos placement-testi osoittaa tason B tai korkeampi,
  oppilas voidaan sijoittaa suoraan Kurssille 3 tai 5. Aiemmat kurssit merkitään
  "ohitettu — voit palata halutessasi" mutta ne eivät ole pakollisia.

### Nopea eteneminen
- 3 perättäistä oppituntia >85% tarkkuudella → näytetään: "Menet hienosti! Hypätäänkö seuraavaan oppituntiin?"
- Kurssin ensimmäinen oppitunti >90% → näytetään: "Tämä vaikuttaa tutulta. Tehdäänkö suoraan kertaustesti?"
- Kertaustesti >95% → kurssi merkitään suoritetuksi, seuraava avautuu välittömästi

### Hidas eteneminen
- Kertaustesti <80%: yksilöity palaute ("Preteriti vs imperfekti -tehtävissä oli vaikeuksia.
  Harjoitellaan lisää ennen etenemistä.") + 3 lisäoppituntia juuri niistä aiheista → uusi testi
- Ei aikarajaa — oppilas ei voi "epäonnistua" kurssilla, vain hidastua

### Kirjoittaminen kaikilla kursseilla
Jokaisella kurssilla on vähintään yksi kirjoitustehtävä. Taso nousee asteittain:
- K1–K2: ei YTL-rubriikki, vain sanallinen palaute
- K3–K5: kevyt YTL-rubriikki (2 dimensiota: sisältö + rakenteet)
- K6–K8: täysi YTL-rubriikki (4 dimensiota, pistemäärä 0–20)

## 5. Opetussivut (Teaching Pages)

Ennen jokaista UUTTA kielioppiaihetta oppilas näkee opetussivun. Rakenne on aina sama:

```
# [Otsikko — lyhyt ja konkreettinen, esim. "Preteriti — yksittäiset menneet teot"]

[1 kappale, max 80 sanaa, selkokielinen suomi. Ei kielitiedejargonia.]

## Muodostus

| Persoona | -ar (hablar) | -er (comer) | -ir (vivir) |
|----------|-------------|-------------|-------------|
| yo       | hablé       | comí        | viví        |
| tú       | hablaste    | comiste     | viviste     |
| él/ella  | habló       | comió       | vivió       |
| nosotros | hablamos    | comimos     | vivimos     |
| vosotros | hablasteis  | comisteis   | vivisteis   |
| ellos    | hablaron    | comieron    | vivieron    |

## Esimerkki

> Ayer **fui** al cine con mis amigos. (Eilen menin elokuviin kavereideni kanssa.)
> El año pasado **vivimos** en Madrid. (Viime vuonna asuimme Madridissa.)

## YO-vinkki 💡

[1–2 lausetta siitä, mitä YO-koe nimenomaan testaa tästä aiheesta.
Esim: "YO-kokeessa preteriti esiintyy usein kertomuksissa. Yleisimmät epäsäännölliset
(ser/ir → fui, estar → estuvo, tener → tuvo) tulevat lähes joka kokeessa."]
```

Promptissa käytetään `puheo-ai-prompt`-skillin system message -pohjaa + `promptFocus`-kenttää
`lib/learningPath.js`:stä.

## 6. Placement-testi — parannukset

### Nykyinen ongelma
20 hardcoded MCQ-kysymystä → oppilas voi arvata 25% oikein per kysymys → placement epäluotettava.

### Parannettu placement

**Vaihe 1 — Taustatiedot (ennen testiä):**
- "Mikä on viimeisin arvosanasi espanjasta koulun kokeessa?" → I / A / B / C / M / E / L / En ole saanut arvosanaa
  *(Huom: kysytään KOULUN arvosanaa, ei YO-arvosanaa — suurin osa ei ole vielä kirjoittanut)*
- "Mikä on tavoitearvosanasi yo-kokeessa?" → sama asteikko
- "Mikä tuntuu vaikeimmalta?" → multi-select: Verbien taivutus / Aikamuodot / Ser vs estar /
  Subjunktiivi / Sanasto / Kirjoittaminen / Luetun ymmärtäminen
- "Kuinka kauan voit harjoitella päivässä?" → 10 min / 20 min / 30 min / 45 min+

**Vaihe 2 — Adaptiivinen testi (8 kysymystä):**
- Käytetään olemassa olevaa `selectDiagnosticQuestions(reportedGrade, 8)` mutta PARANNETAAN:
  - Välitön palaute jokaisen kysymyksen jälkeen (ei vasta lopussa)
  - Vastausaika mitataan — alle 5 sekuntia per kysymys → todennäköisesti arvailu →
    `placement_confidence: "low"` → aloitetaan yksi taso alempaa
  - Jos oppilas saa A-tason kysymykset oikein mutta B-tason väärin → ankkuri B-alkuun
  - Jos oppilas saa kaikki oikein → lisätään 2 M-tason lisäkysymystä

**Vaihe 3 — AI-arviointi (server-side, cachettaan käyttäjälle):**
Prompt: "Oppilas vastasi seuraaviin kysymyksiin... [tulokset]. Hän raportoi koulun arvosanaksi [X]
ja tavoitteekseen [Y]. Hän ilmoitti vaikeuksiksi: [Z]. Kirjoita 2–3 lausetta suomeksi
tutori-äänellä: missä hän on nyt ja mistä aloitetaan. Älä mainitse 'tasoa' abstraktisti —
sano konkreettisesti mitä hänen pitää harjoitella ensin."

**Vaihe 4 — Suositeltava lähtöpiste:**
- `scorePlacementTest()` → `placementLevel`
- Kurssisijoitus: A-taso → Kurssi 1, B-taso → Kurssi 3, C-taso → Kurssi 5, M+ → Kurssi 7
- Jos `placement_confidence: "low"` → yksi kurssi alemmaksi
- Jos tavoite on paljon korkeampi kuin nykyinen taso → näytetään realismia:
  "Tavoitteesi on E, mutta testisi viittaa B-tasoon. Se on täysin saavutettavissa —
  tarvitset noin 60–80 tuntia harjoittelua. Aloitetaan B-tasolta."

## 7. Tutoriääni — päivittäinen coaching

### Dashboard-tervehdys (generoidaan AI:lla, cachettaan 24h)
EI pelkkä "Iltaa, [nimi]." Tutorin viesti on 1–2 lausetta joka ottaa huomioon:
- Edellisen session aihe ja tulos
- Nykyinen kurssi ja oppitunti
- Päivien määrä kokeeseen
- Heikoin tunnistettu osa-alue

Esimerkkejä (puheo-finnish-voice -tyyliin):
- "Hyvä putki! Eilen preteriti sujui hyvin — tänään mennään imperfektiin."
- "Subjunktiivi tuottaa vielä haasteita. Tänään harjoitellaan sitä lisää ennen etenemistä."
- "152 päivää kokeeseen. Olet kurssilla 3/8 — hyvää vauhtia."
- "Pitkä tauko — mutta olet takaisin! Kerrataan ensin viime kerran aihe."

### Session jälkeinen palaute (kaikissa modeissa, ei vain kirjoittamisessa)
Nykyään tutorin palaute tulee vain kirjoittamisessa. Laajennetaan kaikkiin moodeihin:

**Sanasto/kielioppi:**
"[X/Y oikein]. [Jos <70%: Mainitse tarkalleen mikä rakenne tuotti virheitä.]
[Jos >90%: Lyhyt kannustus + vihjaa seuraavaan aiheeseen.]"
Max 2 lausetta. Ei geneeristä "Hyvää työtä!".

**Luetun ymmärtäminen:**
"Ymmärsit tekstin pääkohdat hyvin. [Mainitse yksi sana/rakenne joka esiintyi ja jota kannattaa muistaa.]"

## 8. Tietokantataulut (uudet)

```sql
-- Kurssimääritelmät (seed kerran)
CREATE TABLE curriculum_kurssit (
  key            text PRIMARY KEY,   -- 'kurssi_1' .. 'kurssi_8'
  title          text NOT NULL,
  description    text,
  level          text NOT NULL,      -- 'A'|'B'|'C'|'M'|'E'
  vocab_theme    text,               -- VALID_VOCAB_TOPICS arvo
  grammar_focus  text[],             -- LEARNING_PATH key-arvoja
  lesson_count   int NOT NULL,
  sort_order     int NOT NULL
);

-- Oppitunnit per kurssi (seed kerran)
CREATE TABLE curriculum_lessons (
  id             serial PRIMARY KEY,
  kurssi_key     text REFERENCES curriculum_kurssit(key),
  sort_order     int NOT NULL,
  type           text NOT NULL,      -- 'vocab'|'grammar'|'reading'|'writing'|'mixed'|'test'
  focus          text NOT NULL,      -- esim. "preteriti + matkailusanasto"
  exercise_count int NOT NULL DEFAULT 8,
  teaching_snippet text              -- 2-3 lausetta näytetään ennen oppituntia
);

-- Opetussivut (generoidaan AI:lla, cachettaan)
CREATE TABLE teaching_pages (
  topic_key      text PRIMARY KEY,
  content_md     text NOT NULL,
  generated_at   timestamptz DEFAULT now()
);

-- Käyttäjän edistyminen kurssilla
CREATE TABLE user_curriculum_progress (
  user_id        uuid REFERENCES auth.users(id),
  kurssi_key     text REFERENCES curriculum_kurssit(key),
  lesson_index   int NOT NULL,
  completed_at   timestamptz DEFAULT now(),
  score_correct  int,
  score_total    int,
  PRIMARY KEY (user_id, kurssi_key, lesson_index)
);

-- Placement-tulokset (parannettu)
ALTER TABLE users ADD COLUMN IF NOT EXISTS
  self_reported_grade     text,      -- koulun arvosana I..L
  target_grade            text,      -- tavoite I..L
  weak_areas              text[],
  daily_goal_minutes      int DEFAULT 20,
  placement_confidence    text,      -- 'high'|'low'
```

- Lisää vitest joka vertaa `USER_TABLES`:ia skeemaan → drift ei toistu.
- Verifioi: tee testikäyttäjä → "poista tili" → 0 residuaaliriviä missään user_id-taulussa (SQL-todiste).

### 3. P1-4 — RLS initplan -migraatio (skaala)
- Migraatio (MCP `apply_migration`): pudota+luo `free_usage`-policyt (`_owner_select/_owner_upsert/_owner_update`) ja `translation_accepted_user_insert` `(select auth.uid())`-muodolla.

### 4. P1-3 — RLS-safety-net kytkentä (TÄYSI, päätös 2 yllä)

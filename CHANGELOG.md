# Changelog

## [1.0.0] - 2026-04-17

### Added
- Vocabulary exercises with adaptive difficulty (levels I-L)
- Grammar drills with 7 topic areas and 4 exercise types
- Reading comprehension with AI-generated texts
- Writing tasks graded against YTL criteria
- Mock exam mode (45 min, reading + writing)
- User authentication (email/password via Supabase)
- Dashboard with progress chart, heatmap, streak counter
- Spaced repetition for wrong answers
- Exercise bank to reuse AI-generated exercises
- Supabase-backed rate limiting and caching for serverless
- Report exercise button for quality control
- Admin endpoint for flagged exercises
- Pro/Free tier with LemonSqueezy payments
- Desktop sidebar + mobile bottom nav
- Keyboard shortcuts for all exercise modes
- ES module architecture (13 modules)
- Security headers via helmet
- CI pipeline (GitHub Actions)
